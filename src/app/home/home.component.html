Qualcosa su Questo Autonoleggio

Database Relazionale con MySQL

  • Ho usato il Database Autonoleggio usato in classe per la realizzazione di questo progetto
  • Il database si trova in Localhost, alla porta di Default 3306
  • Le query non vengono fatte direttamente al DB, ma vengono gestite attraverso un Back End
  • Ho realizzato questa Web-App per poter gestire il Database e fare le operazioni CRUD (Create, Read, Update, Delete) su questo Database da remoto, una volta rilasciato su un hosting specifico

Backend con Express

  • Mi sono affidato a una Guida Online per la Realizzazione del Backend, un server basato sulla libreria Express e Router, in Node.js
  • Il Server si trova in Localhost alla porta 8080, che gestisce le chiamate al DB in base alle chiamate che riceve dal FrontEnd, con l'uso di Promise e richieste Asincrone
  • La libreria router decide le chiamate da fare al DB in base al tipo di richiesta (GET, PUT, POST o DELETE) e all'indirizzo richiesto (Esempio: "localhost:8080/categoria/10"), facendo quindi al DB la query conseguente
  • Il risultato della Query, poi, viene tradotto in un pacchetto Json e Mandato al Frontend, sfruttando il Server-Service scritto ad hoc in Angular
  • Non ho realizzato, sebbene sia piuttosto semplice, la possibilità di scrivere o generare direttamente la Query nel Front End, per evitare rischi di Sicurezza quali SQL-Injection. Avrei potuto realizzare una semplice richiesta POST con la Query come parametro, ma avrebbe aperto la strada ad attacchi al DB

Frontend con Angular e BootStrap

  • Ho scelto di usare Angular e Bootstrap perchè già trattati in un modulo precedente e perchè mi permettono di creare velocemente Layout Responsive e Siti Web dinamici, integrandosi molto bene col Back-End Express
  • In base al Json inviato dal Backend e ricevuto dal Server-Service, infatti, è possibile creare interfacce che si Auto-Generano sulla base dei dati ricevuti
  • Un controllo delle operazioni avviene direttamente nel Frontend, per esempio nascondendo il pulsante Elimina dalle Categorie con Auto presenti , oppure controllando la tipologia dei dati inseriti nei Form
  • Inoltre, con questa tecnologia, riduco l'affaticamento del Backend e del Database, in quanto l'elaborazione grafica è realizzata localmente nel client

Istruzioni Per L'Uso

Dove Trovare il codice

Ho messo online il codice all'indirizzo https://github.com/Ciull0/DbDanieleCiulli

Il progetto poi dovrà essere scaricato (o clonato il repository) per essere eseguito localmente


Come Avviare il Progetto

  1. Verifica che il Database MySQL sia attivo, e controlla la porta, utente e password del db nel file index.js nella cartella ProgettoDatabase/server/src ProgettoDatabase/server/src
  2. Apri un Command Prompt nella cartella ProgettoDatabase\server\src ed esegui il comando node index.js. Questo Aprirà il Server del Db sulla porta 8080 del localhost usando Node.js
  3. Con un secondo Command Prompt, vai nella cartella ProgettoDatabase e, con Angular10 installato, esegui il comando ng serve --open. Al termine della compilazione di Angular, si aprirà una finestra del browser con il progetto visibile all'indirizzo localhost:4200.

Dove può trovare il codice MySql?

Il codice MySQL è nella del file events.js nella cartella server/src

Sono rapidamente accessibili qui: https://github.com/Ciull0/DbDanieleCiulli/blob/main/server/src/events.js
